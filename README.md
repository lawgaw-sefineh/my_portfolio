# 🌐 My Portfolio Website

Welcome to my personal portfolio website! This project is built completely using pure HTML5 to showcase my web structure skills and personal projects.

## 🚀 Features
* **Pure HTML5 Structure:** Built entirely with semantic HTML5 elements.
* **Clean Layout:** Organized sections for easy navigation and readability.
* **Projects Section:** Showcases my recent work and development links.
* **Contact Information:** Simple ways to reach out to me.

## 🛠️ Tech Stack Used
* **HTML5 Only:** Used for structures, headers, links, and project images.

## 📂 Project Structure
```text
├── index.html          # Main web page
└── images/             # Folder for website images and graphics
```

## 💻 How to Run Locally
1. Clone this repository or download the files.
2. Double-click the `index.html` file to open it in any web browser.
